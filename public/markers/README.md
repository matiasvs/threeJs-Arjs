# AR Markers

## Marcador Hiro

El archivo `hiro.png` contiene el marcador Hiro, que es el marcador estándar de AR.js.

### Cómo usar:

1. **Imprime el marcador**: Descarga e imprime `hiro.png` en papel blanco
2. **Tamaño recomendado**: Imprime a un tamaño de al menos 10x10 cm para mejor detección
3. **Iluminación**: Asegúrate de tener buena iluminación sin reflejos
4. **Distancia**: Mantén el marcador a 20-50 cm de la cámara

### Acceder al marcador:

Cuando ejecutes la aplicación, puedes acceder al marcador en:
- Desarrollo: `http://localhost:5173/markers/hiro.png`
- Producción: `https://<tu-usuario>.github.io/threejs-arjs/markers/hiro.png`

### Crear marcadores personalizados:

Puedes crear tus propios marcadores usando el generador oficial:
https://ar-js-org.github.io/AR.js/three.js/examples/marker-training/examples/generator.html
